Usage: insforge [options] [command]

InsForge CLI - Command line tool for InsForge platform

Options:
  -V, --version                output the version number
  --json                       Output in JSON format
  --api-url <url>              Override Platform API URL
  -y, --yes                    Skip confirmation prompts
  --reason <text>              Agent: what the operation does and why (intent) —
                               shown to the human approver for destructive
                               operations
  --impact <text>              Agent: implications — who/what is affected, data
                               loss, reversibility — shown on the approval page
  --recommendation <text>      Agent: your recommendation to the human approver
  --flag-destructive [reason]  Agent: flag this op as destructive for human
                               approval even if InsForge's rules consider it
                               safe (escalate-only — cannot downgrade the
                               verdict)
  -h, --help                   display help for command

Commands:
  login [options]              Authenticate with InsForge platform
  logout                       Log out from InsForge platform
  whoami                       Show current authenticated user
  create [options]             Create a new InsForge project
  current                      Show current CLI context (user, org, project)
  list                         List all organizations and their projects
  docs [feature] [language]    Browse InsForge SDK documentation
  link [options]               Link current directory to an InsForge project (no
                               args: installs agent skills only)
  orgs                         Manage organizations and members
  projects                     Manage projects
  branch                       Manage backend branches
  db                           Database operations
  functions                    Manage edge functions
  storage                      Manage storage
  deployments                  Deploy and manage frontend sites
  secrets                      Manage secrets
  logs [options] <source>      Fetch backend container logs (insforge.logs |
                               postgREST.logs | postgres.logs | function.logs |
                               function-deploy.logs)
  metadata                     Show backend metadata (auth, database, buckets,
                               edge functions, realtime, AI models)
  diagnose [options]           Backend diagnostics — run with no subcommand for
                               a full health report
  payments                     Manage payments
  billing                      Inspect subscription, plan, and credits
  usage [options]              Show organization usage for the current billing
                               period
  backups                      Manage project backups
  compute                      Manage compute services (Docker containers on
                               Fly.io)
  posthog                      Manage PostHog product analytics integration
  webscraper                   Manage web scraper integrations
  ai                           Manage AI model gateway setup
  domains                      Register and attach custom domains
  memory                       Store and recall durable agent memories
  schedules                    Manage scheduled tasks (cron jobs)
  config                       Manage insforge.toml (declarative project
                               configuration)
  help [command]               display help for command
